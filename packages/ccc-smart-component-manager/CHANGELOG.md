## [1.3.0] - 2025-07-21 ✅ 当前版本 - 组件拖拽排序功能

### 🎉 新增 (Added)
- ✅ **组件拖拽排序功能**: 选中单一节点时，可通过拖拽调整组件在节点上的顺序
  - ✅ 智能检测：只在选中单一节点时启用拖拽功能，多选时自动禁用
  - ✅ 视觉反馈：拖拽过程中的动画效果、状态提示和拖拽光标
  - ✅ 安全机制：拖拽时禁用点击删除，防止误操作
  - ✅ 实时更新：拖拽完成后自动刷新组件列表显示新顺序

### 🔧 改进 (Changed)
- ✅ **组件按钮交互**: 增强了组件按钮的交互体验
  - ✅ 单节点选择时显示拖拽光标（move cursor）
  - ✅ 拖拽状态下智能禁用点击事件
  - ✅ 改进的悬停效果和拖拽目标高亮

### 🛠️ 技术实现 (Technical)
- ✅ **拖拽API**: 新增组件顺序调整相关的场景脚本方法
  - ✅ `move-component`: 移动组件到指定位置的核心方法
  - ✅ `get-component-indices`: 获取组件索引信息的辅助方法
  - ✅ `component-moved`: 移动完成通知机制
- ✅ **UI增强**: 新增拖拽相关的CSS样式和动画效果
  - ✅ `.draggable`: 可拖拽状态样式
  - ✅ `.dragging`: 拖拽进行中样式
  - ✅ `.drag-over`: 拖拽悬停目标样式
  - ✅ `.drag-hint`: 拖拽提示信息样式
- ✅ **状态管理**: 完善的拖拽状态跟踪和错误处理

### 📚 文档 (Documentation)
- ✅ **使用指南**: 新增组件拖拽排序功能的详细使用文档
- ✅ **演示说明**: 提供完整的功能演示和测试步骤
- ✅ **技术文档**: 详细的技术实现说明和API文档

### 🎯 实现方案

#### 核心算法设计

1. **节点树遍历算法**
   ```javascript
   function traverseNodeTree(rootNode, callback) {
       callback(rootNode);
       if (rootNode.children) {
           rootNode.children.forEach(child => {
               traverseNodeTree(child, callback);
           });
       }
   }
   ```

2. **属性解析算法**
   ```javascript
   function parseComponentProperties(component) {
       const properties = [];
       const constructor = component.constructor;
       const className = cc.js.getClassName(constructor);

       // 解析@property装饰器定义的属性
       if (constructor.__props__) {
           constructor.__props__.forEach(prop => {
               if (prop.type && cc.js.isChildClassOf(prop.type, cc.Component)) {
                   properties.push({
                       name: prop.name,
                       type: prop.type,
                       currentValue: component[prop.name]
                   });
               }
           });
       }
       return properties;
   }
   ```

3. **名称匹配算法**
   ```javascript
   function matchNodeName(propertyName, nodeName, options = {}) {
       const { caseSensitive = false, camelCaseConversion = true } = options;

       // 精确匹配
       if (caseSensitive ? propertyName === nodeName :
           propertyName.toLowerCase() === nodeName.toLowerCase()) {
           return { match: true, score: 100, type: 'exact' };
       }

       // 驼峰命名转换匹配 (btnOk -> btn_ok)
       if (camelCaseConversion) {
           const convertedName = propertyName.replace(/([A-Z])/g, '_$1').toLowerCase();
           if (convertedName === nodeName.toLowerCase()) {
               return { match: true, score: 90, type: 'camelCase' };
           }
       }

       return { match: false, score: 0, type: 'none' };
   }
   ```

4. **自动挂载执行算法**
   ```javascript
   function autoMountProperties(component, nodeTree) {
       const properties = parseComponentProperties(component);
       const mountResults = [];

       properties.forEach(prop => {
           if (prop.currentValue !== null) return; // 已挂载，跳过

           traverseNodeTree(nodeTree, (node) => {
               const matchResult = matchNodeName(prop.name, node._name);
               if (matchResult.match) {
                   const targetComponent = node.getComponent(prop.type);
                   if (targetComponent) {
                       component[prop.name] = targetComponent;
                       mountResults.push({
                           property: prop.name,
                           node: node._name,
                           component: cc.js.getClassName(prop.type),
                           success: true
                       });
                   }
               }
           });
       });

       return mountResults;
   }
   ```
